# 🧪 Experiment Report Skill

一个通用的 AI 实验报告生成工作流。你只需要描述实验内容，它帮你完成：

**实验实现 → 前端可视化 → 截图 → 实验报告**

## ✨ 它能做什么

- 🔬 **自动实现实验** — 描述你的实验，AI 帮你跑通
- 🖥️ **生成前端展示** — 可视化页面，策略图、曲线图、对比图
- 📸 **截图入库** — 前端截图直接进报告，真实数据不造假
- 📝 **生成实验报告** — Markdown 格式，可直接转 Word/docx

## 🚀 怎么用

1. 在你的 AI 工具中安装这个 Skill
2. 告诉它你要做什么实验（比如："我要做 SARSA 和 Q-learning 的对比实验"）
3. 等它跑完实验 → 生成前端 → 确认效果 → 自动写报告

就这么简单。

```

```

## 🎯 适用场景

- 强化学习实验报告
- 机器学习算法对比
- 数据分析报告
- 任何需要"跑实验 + 展示结果 + 写报告"的作业

## 🔧 核心原则

| 原则 | 说明 |
|------|------|
| 数据真实 | 所有图表必须来自实际运行结果，不许伪造 |
| 前端先行 | 先做可视化页面让你确认，再写报告 |
| 报告规范 | 贴近 docx 结构，方便直接转 Word |
| 心得真实 | 最后一章做人味化处理，不像 AI 写的 |

## 📄 报告模板结构

1. 实验名称
2. 实验目的
3. 实验环境与任务定义
4. 算法/方法原理
5. 实验实现
6. 实验结果
7. 结果分析
8. 实验结论
9. 心得感悟
10. 附录
